# DataSense

```sh
npm i datasense
```

## 背景

DataSense 是个管理数据及数据流的一个简单、高效、实用、有趣的前端开源库，包括访问者模型、绑定与订阅关系、时序任务控制等，通过这些基础功能，可以更快速地实现上层业务逻辑，或者在实现用于其它业务场景的高阶技术封装时，基于此进行二次开发。
