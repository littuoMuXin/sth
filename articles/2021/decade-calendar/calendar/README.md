以上表格为十进制历的通用日历，适用于任意一年。其中第一列为拾份，后续为其内所有日期，最后一行的星号（`*`）在平年时为空，闰年时为6。从这个表格可以看得出，绝大多数内容为简单的重复，因此实际此历法实际并不需要日历，非常实用。
